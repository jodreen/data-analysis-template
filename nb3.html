Step 1: Outline¶

We are going to use the cleaned data to ascertain whether or not there is any significant correlation between the volume of search data for a particular publicly traded company and its share price. First, we need to write a function to take all of the gathered data and find the correlation between the GTrends value and the stock price, for each company in the Dow Jones and the S&P 500 indices.

Note: All analysis conclusions will be made in NB4_project_report

Step 2: Making the Function to Get All the Data

Since we gathered raw GTrends data for all companies and more data for how the companies' search volume changed when positive and negative terms were appended to the search, we need to separate the data into various categories for the type of search, as well. Let's write a function that can achieve all of this in one go.

%%R
##creating a list of files in the folder
list_of_files <- list.files("~/Desktop/googletrends-stocks/notebooks/data/cleaned/DowJones_Modified")

#creating the lists to store the correlations
normal_cor_list <- c()
percent_cor_list <- c()
sell_cor_list <- c()
ticker_cor_list <- c()
buy_cor_list <- c()
my_str = "data/cleaned/DowJones_Modified/"

##creating a function to gather all the data

get_all_data = function(x) {
    if (x == 'SP500') {
        my_str <<- 'data/cleaned/SP500/'
        list_of_files <<- list.files("~/Desktop/googletrends-stocks/notebooks/data/cleaned/SP500")
    }
    for (s in 1:length(list_of_files)){
        ##opening each file in the folder
        myStock = read.csv(paste(my_str,list_of_files[s],sep = ""))
        myStock = na.omit(myStock)
        growth2 = c(0)
        growth = 0
        ##adding a column with relative stock prices
        for (i in 1:length(myStock$Price)){
            growth = (myStock$Price[i]-myStock$Price[i-1])/myStock$Price[i-1]*100
            growth2= c(growth2,growth)
        }
        if (nrow(myStock) == 0){
            next
        }
        if (ncol(myStock) < 3){
            next
        }
        ##separating key term files
        if (substr(list_of_files[s],1,4) == "sell"){
            sell_cor_list <<- c(cor(myStock[,2], myStock[,3]),sell_cor_list)
            next
        }
        if (substr(list_of_files[s],1,3) == "buy"){
            buy_cor_list <<- c(cor(myStock[,2], myStock[,3]),buy_cor_list)
            next
        }
        if (toupper(substr(list_of_files[s],2,2)) == (substr(list_of_files[s],2,2))) {
            ticker_cor_list <<- c(cor(myStock[,2], myStock[,3]),ticker_cor_list)
            next
        }
        myStockNew = data.frame(myStock, "PriceGrowth"= growth2)
        ##lists of correlatons for absolute and relative stock price data
        normal_cor_list <<-c(cor(myStockNew[,2], myStockNew[,3]),normal_cor_list)
        percent_cor_list <<-c(cor(myStockNew[,2], myStockNew[,4]),percent_cor_list)
        normal_cor_list <<- na.omit(normal_cor_list)
        percent_cor_list <<- na.omit(percent_cor_list)
        ticker_cor_list <<- na.omit(ticker_cor_list)
        sell_cor_list <<- na.omit(sell_cor_list)
        buy_cor_list <<- na.omit(buy_cor_list)
        }
    return()
}

Now that we have a function that puts all of the correlations in lists, let's run the function for all of the companies in the Dow Jones Index.

Step 5: Splitting Up Sectors¶

Now, let's split the companies into different sectors to see whether or not certain sectors are more likely to see their share price fluctuation follow the GTrends data. We did this by manually separating the CSV files using the broad sectors listed here: http://en.wikipedia.org/wiki/List_of_S&P_500_companies.

After organizing the CSV files, we will write code to analyze each sector.

In [35]: 
%%R
get_SP500_sector_data = function(object) {
    list_of_folders <<- list.files("~/Desktop/googletrends-stocks/notebooks/data/sectors")
    list_of_folders2 = paste("~/Desktop/googletrends-stocks/notebooks/data/sectors/",list_of_folders, sep = '')
    correlation_group = c()
    for(i in 1:10) {
        files = paste(sep = '', "~/Desktop/googletrends-stocks/notebooks/data/sectors/", list_of_folders[i], '/',list.files(list_of_folders2[i]))
        correlations = c()
        for(s in 1:length(list.files(list_of_folders2[i]))) {
            x = read.csv(files[s])
            correlations = c(correlations, cor(x[2], x[3])^2)
            }
        correlation_group = c(correlation_group, mean(correlations))
        
    }
    return(correlation_group)
}

Step 6: Company Case Study

Although we tested all of the stocks in the S&P 500 collectively, we found several potential areas for error in our findings.

  1. Many of the ticker names are acronyms for other common names or search terms (ex. ABC is ticker symbol for AmerisourceBergen Corporation)
  2. Many company name searches are infrequent due to shorter nicknames (ex. Goodyear Tire & Rubber)

Thus we decided to deeper analyze 10 different companies. All of these companies have ticker symbols that are unlikely to be mistaken for another search term and represent firms from a variety of industries.

  1. EBAY (eBay Inc.)
  2. IBM (International Business Machines Corporation)
  3. EXPD (Expeditors Internatonal of Washington)
  4. BIIB (Biogen Idec Inc)
  5. XEL (Xcel Energy Inc)
  6. BXP (Boston Properties, Inc.)
  7. AMZN (Amazon.com, Inc.)
  8. LYB (LyondellBasell Industries)
  9. VZ (Verizon Communications Inc.)
  10. CSV (CVS Corporation)
